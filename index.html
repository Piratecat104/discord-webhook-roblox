About This Visualizer

This audio visualizer uses the Web Audio API to analyze microphone input and create visual representations of the sound.

You can choose between different visualization types:

  • Frequency Bars: Displays amplitude across frequency bands
  • Waveform: Shows the audio waveform in real time
  • Circular: A radial visualization that pulses with the sound
  • Particles: Dynamic particles that react to audio frequencies

Adjust the sensitivity and color schemes to customize your experience.
